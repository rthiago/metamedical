<task type="auto">
  <name>Task 1: Create CSS design system with header, nav, footer, and responsive styles</name>
  <files>css/style.css</files>
  <action>
Create `css/style.css` with the complete design system and component styles:

**CSS Custom Properties (`:root`):**
- Primary color: a professional medical blue (e.g., `#0056b3` or similar deep blue)
- Secondary color: a lighter complementary blue (e.g., `#e8f4fd`)
- Accent color: a teal or green-blue for highlights (e.g., `#00897b`)
- Text colors: dark charcoal for body (`#333`), medium gray for secondary text (`#666`)
- Background: white (`#fff`) for content, light gray (`#f8f9fa`) for alternating sections
- Font family: Use a clean sans-serif from Google Fonts — Roboto or Inter for body, Montserrat or Poppins for headings
- Spacing scale: 0.5rem, 1rem, 1.5rem, 2rem, 3rem, 4rem increments
- Max content width: 1200px
- Border radius: 4px for subtle rounding
- Box shadow for cards: subtle shadow like `0 2px 8px rgba(0,0,0,0.1)`

**Reset & Base:**
- Box-sizing border-box on all elements
- Margin/padding reset on body
- Base font-size 16px, line-height 1.6
- Smooth scrolling on html

**Typography:**
- h1: 2.5rem, bold, heading font
- h2: 2rem
- h3: 1.5rem
- p: 1rem, body font
- Links: primary color, no underline by default

**Header & Navigation:**
- Fixed or sticky header with white background and subtle bottom shadow
- Logo area on the left (text placeholder "MetaMedical" styled as brand — will be replaced with image in Phase 2 if logo file is provided)
- Nav links on the right, horizontal layout on desktop
- Active page link gets a visual indicator (bottom border or color change)
- Nav links: Home, Empresa, Produtos, Localização, Contato

**Hamburger Menu (CSS-only, no JavaScript):**
- Use the checkbox hack: hidden checkbox input + label styled as hamburger icon
- On mobile (below 768px), hide the nav links by default
- When checkbox is checked, show nav links as vertical stack
- Hamburger icon: 3 horizontal bars using CSS (span elements or pseudo-elements)
- Animate the hamburger to X when open (CSS transitions on the spans)
- Hide the hamburger and checkbox on desktop (above 768px)

**Footer:**
- Dark background (primary color or dark charcoal)
- White text
- Three-column layout on desktop (company info, navigation links, contact info), stacking on mobile
- Contains: "MetaMedical" name, CNPJ: 21.033.671/000129, Av. Dos Autonomistas, 896 - Osasco/SP, RE 8365-SP
- Copyright line at bottom

**Layout Utilities:**
- `.container` class: max-width 1200px, centered with auto margins, padding 0 1rem
- `.section` class: padding 4rem 0
- `.section--alt` for alternating background color

**Responsive Breakpoints:**
- Mobile: up to 767px
- Tablet: 768px to 1023px
- Desktop: 1024px+
- Use mobile-first approach (base styles are mobile, media queries add desktop)

Do NOT use any JavaScript. The hamburger menu MUST work with CSS only (checkbox hack).
